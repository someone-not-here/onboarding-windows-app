### Why is code formatting important?
Taking time to format code properly helps maintain the codebase's readability and consistency. This puts ease on other developers who have to read through and review the code.

### What issues did the linter detect?
ESLint detected:
- Unused variables
- Unused functions

### Did formatting the code make it easier to read?
While the pre-formatted code was readable, it was much more consistent in style after formatting, which contributes to readability.
